## üéØ Content Writing Guidelines

### For Each Article:
1. **Clear title** - Descriptive and searchable
2. **Introduction** - What will user learn?
3. **Step-by-step instructions** - Use numbered lists
4. **Screenshots** - Add visual aids where helpful
5. **Admonitions** - Use `:::note`, `:::tip`, `:::warning`, `:::danger`
6. **Code examples** - For API documentation
7. **Related articles** - Cross-link related content
8. **Concise** - Keep focused on one topic

### Writing Style:
- Use second person ("you") - User-focused
- Active voice - "Click the button" not "The button should be clicked"
- Clear and simple language
- Short paragraphs (2-4 sentences)
- Bullet points for lists
- Numbered steps for procedures
