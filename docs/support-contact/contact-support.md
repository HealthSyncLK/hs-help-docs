## Before Contacting Support

### Gather Information

Help us help you faster by providing:
- Account email address
- Description of the issue
- What you were trying to do
- Error messages (exact text or screenshot)
- Device and browser information
- Steps to reproduce the problem

### Check Resources First

1. **Search Help Documentation**
   - Search this help center
   - Most answers found here
   - Faster than waiting for response

2. **Check System Status**
   - [status.healthsync.com](https://status.healthsync.com)
   - Ongoing incidents posted
   - Scheduled maintenance listed

3. **Try Troubleshooting Steps**
   - Clear browser cache
   - Try different browser
   - Update mobile app
   - Restart device
   - Check internet connection

4. **Review FAQs**
   - [Common Questions](../troubleshooting-faqs/faq.md)
   - [Troubleshooting Guide](../troubleshooting-faqs/intro.md)
