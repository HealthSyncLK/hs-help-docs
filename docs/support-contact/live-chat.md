## Ending Chats

### When Resolved
Agent will ask:
- "Is there anything else I can help with?"
- "Does that solve your issue?"
- Rate your experience
- Receive transcript via email

### If You Need to Leave
- Tell agent you need to go
- They'll document progress
- Option to schedule follow-up
- Conversation saved

### Idle Timeout
Inactive for 10 minutes:
- Warning after 8 minutes
- Chat ends after 10 minutes
- Transcript emailed
- Can start new chat anytime

## Common Chat Questions

**Q: Can I request a specific agent?**
A: Not usually, but Premium customers can request their account manager.

**Q: Is chat recorded?**
A: Yes, for quality assurance and training. Transcripts available to you.

**Q: Can multiple people join one chat?**
A: No, but you can add others to email ticket after chat.

**Q: What if chat disconnects?**
A: Start new chat; reference previous chat number to continue.

**Q: Can I use chat on smartwatch?**
A: Not currently; use phone, tablet, or computer.
