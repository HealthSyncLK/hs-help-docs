# User Roles & Permissions

Understand different user roles and their permissions in HealthSync.

## User Roles

### Account Owner
- Full access to all features
- Manage billing and subscription
- Add/remove family members
- Control all sharing permissions

### Family Member
- Access to own records
- Limited sharing capabilities
- Can view shared records from other family members

### Guest User (Shared Access)
- View-only access to specific records
- Cannot modify or delete records
- Access can be revoked at any time

## Permission Levels

### View Only
Can see records but cannot edit or delete them.

### Edit Access
Can view and modify records but cannot delete them.

### Full Access
Complete control including viewing, editing, and deleting records.
