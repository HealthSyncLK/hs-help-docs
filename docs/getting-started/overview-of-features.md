# Overview of Features

Explore the comprehensive features available in HealthSync.

## Core Features

### Health Records Management
- Upload and organize medical documents
- Categorize records by type
- Tag and search functionality

### AI Report Analyzer
- Automatic analysis of medical reports
- Plain language summaries
- Key findings highlights

### Family Health Management
- Manage multiple family member profiles
- Shared family health dashboard
- Granular permission controls

### Secure Sharing
- Share records with healthcare providers
- Time-limited access links
- Audit trail of all access

## Premium Features

Available with paid subscriptions:
- Unlimited AI analyses
- Advanced analytics and insights
- Priority support
- Extended storage

## Mobile App Features

Access HealthSync on the go:
- Full feature parity with web app
- Offline access to records
- Push notifications
- Document scanning
