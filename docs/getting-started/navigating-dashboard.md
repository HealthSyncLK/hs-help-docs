# Navigating the Dashboard

Understand the main components of your HealthSync dashboard.

## Dashboard Overview

Your dashboard provides quick access to:
- Recent health records
- Upcoming appointments
- AI analysis results
- Quick action buttons

## Main Navigation

### Top Menu Bar
- Home
- Records
- Family
- AI Analyzer
- Settings

### Sidebar Navigation
Access different modules from the collapsible sidebar.

## Quick Actions

Use quick action buttons to:
- Upload new records
- Schedule appointments
- Contact support
- Run AI analysis

## Search Functionality

Use the global search to find any record, document, or information quickly.
