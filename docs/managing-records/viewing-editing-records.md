# Viewing & Editing Records

Manage and organize your health records effectively.

## Viewing Records

### List View
Browse all records in a chronological list with filters and sorting options.

### Grid View
Visual thumbnail view for quick identification of documents.

### Timeline View
See your health history on an interactive timeline.

## Record Details

Each record displays:
- Document preview
- Upload date and source
- Categories and tags
- AI analysis results (if available)
- Sharing status

## Editing Records

### Editable Fields
- Title and description
- Categories and tags
- Date of service
- Healthcare provider information

### Document Replacement
Replace documents while maintaining the record history.

## Search and Filter

Quickly find records using:
- Text search
- Date ranges
- Categories
- Tags
- Healthcare providers
