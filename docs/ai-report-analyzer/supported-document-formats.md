# Supported Document Formats

Learn which document types and formats work with AI analysis.

## Supported File Types

### Documents
- **PDF** - Most common format for medical reports
- **DOCX** - Microsoft Word documents
- **TXT** - Plain text files

### Images
- **JPEG/JPG** - Scanned documents and photos
- **PNG** - High-quality scans
- **HEIC** - iPhone photos
- **TIFF** - Medical imaging

### Medical Standards
- **HL7** - Health Level Seven messages
- **FHIR** - Fast Healthcare Interoperability Resources
- **DICOM** - Medical imaging standard (metadata extraction)

## Document Categories Best Suited for AI Analysis

### Highly Accurate
- Lab test results
- Blood work reports
- Pathology reports
- Radiology reports

### Good Accuracy
- Discharge summaries
- Visit notes
- Prescription records
- Vaccination records

### Limited Support
- Handwritten notes (OCR dependent)
- Low-quality scans
- Non-English documents
- Complex imaging reports

## File Size Limits

- Free plan: Up to 10MB per file
- Premium plan: Up to 50MB per file
- Batch processing: Up to 100 files at once
