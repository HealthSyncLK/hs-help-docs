# Fixing Errors in Analysis

Learn how to handle and correct AI analysis errors.

## Common Analysis Issues

### Text Extraction Errors
- Poor image quality
- Handwritten text
- Complex layouts

**Solution**: Re-upload with better quality or use PDF format.

### Incorrect Value Detection
- Misread numbers
- Wrong units of measurement
- Missed decimal points

**Solution**: Manually edit the extracted values.

### Misclassified Results
- Wrong test type identification
- Incorrect categorization

**Solution**: Correct the classification manually.

## Manual Corrections

### Editing AI Results
1. Click "Edit Analysis" on the summary
2. Modify incorrect information
3. Add notes explaining the correction
4. Save changes

### Re-running Analysis
If the analysis failed or is incomplete:
1. Go to document view
2. Click "Re-analyze Document"
3. Wait for new analysis
4. Compare results

## Improving Document Quality

### Before Upload
- Ensure good lighting for photos
- Use high-resolution scans
- Flatten multi-page documents
- Remove unnecessary pages

### File Optimization
- Use PDF format when possible
- Compress large files without quality loss
- Ensure text is selectable in PDFs

## Reporting Issues

Help us improve:
1. Click "Report Issue" on the summary
2. Describe the problem
3. Attach the original document (optional)
4. Submit feedback
