# Offline Access

Access your health records even without an internet connection.

## How Offline Access Works

### What's Available Offline
- Previously downloaded documents
- Cached record metadata
- Saved search results
- Settings and preferences
- Family member profiles
- Recently viewed pages

### What Requires Internet
- Uploading new documents
- Running AI analysis
- Accessing non-downloaded documents
- Real-time sync
- Sharing documents
- Adding family members

## Setting Up Offline Access

### Automatic Downloads

**Smart Download (Default)**
Automatically downloads:
- Recently viewed documents (last 30 days)
- Frequently accessed records
- Starred/favorited documents
- Essential medical records

**Configure in Settings**
Mobile: Settings > Offline Access > Auto-download
- Enable/disable automatic downloads
- Set time range (7, 30, 90 days)
- Choose categories to auto-download

### Manual Downloads

#### Download Individual Documents
1. Open document
2. Tap download icon (⬇️)
3. Wait for download to complete
4. Checkmark indicates offline availability

#### Bulk Download
1. Go to Records view
2. Select multiple documents
3. Tap "..." menu
4. Choose "Make Available Offline"
5. Confirm download

#### Download Entire Categories
1. Go to Settings > Offline Access
2. Tap "Download by Category"
3. Select categories (Lab Results, Prescriptions, etc.)
4. Tap "Download Selected"

## Managing Offline Storage

### Check Storage Usage
Settings > Offline Access > Storage

View:
- Total offline documents
- Storage used
- Available space
- Breakdown by category

### Storage Limits

**Free Plan**
- Up to 100MB offline storage
- ~50-100 documents (depending on size)

**Plus/Premium Plans**
- Up to 2GB offline storage
- ~1,000+ documents

### Freeing Up Space

**Remove Individual Downloads**
1. Open document
2. Tap download icon (with checkmark)
3. Confirm removal
4. Document still accessible online

**Clear All Offline Data**
Settings > Offline Access > Clear Offline Storage
- Removes all downloads
- Frees up maximum space
- Can re-download anytime

**Smart Cleanup**
Settings > Offline Access > Smart Cleanup
- Automatically removes old downloads
- Keeps frequently accessed documents
- Runs weekly

## Offline Features

### View Documents
- Read downloaded documents
- Zoom and pan images
- View AI summaries (if already generated)
- Navigate between documents

### Search
- Search downloaded documents only
- Filters work on cached data
- Results limited to offline content

### Edit Metadata
- Edit document titles
- Add tags and notes
- Change categories
- Changes upload when online

### Queue Actions
Actions queued for when online:
- Document uploads
- Sharing requests
- AI analysis requests
- Settings changes

View queue: Settings > Offline Access > Pending Actions

## Offline Indicators

### Visual Cues
- **Cloud with checkmark**: Available offline
- **Cloud outline**: Online only
- **Spinning arrow**: Downloading
- **Offline badge**: Currently offline

### Offline Mode Banner
When offline, banner appears:
- "You're offline. Some features unavailable."
- Shows pending actions count
- Tap to view queue

## Working Offline

### Best Practices

**Before Going Offline**
1. Download important documents
2. Complete any pending edits
3. Check sync status
4. Verify critical records available

**While Offline**
- View and review documents
- Add notes and tags
- Prepare uploads (queue for later)
- Search downloaded content

**When Back Online**
- Queued actions auto-execute
- New downloads sync
- View sync summary notification

### Limitations

**Cannot Do Offline:**
- Upload new documents (queued)
- Run AI analysis (queued)
- Share documents (queued)
- Add family members (queued)
- Access online-only documents
- Receive real-time updates

## Troubleshooting

### Documents Won't Download

**Check:**
- Internet connection active
- Sufficient storage space
- Not in offline mode
- File not corrupted

**Try:**
- Retry download
- Free up storage
- Update app
- Contact support if persists

### Offline Mode Not Working

**Verify:**
- Documents actually downloaded (checkmark icon)
- Offline mode properly configured
- Not cleared cache recently
- App storage permissions granted

### Sync Issues After Being Offline

**Solutions:**
1. Manual sync: Settings > Sync Now
2. Review pending queue
3. Check for conflicts
4. Retry failed items
5. Contact support if stuck

## Platform-Specific Notes

### iOS
- Downloads stored in app sandbox
- Included in iCloud backup (if enabled)
- Cleared if app deleted
- Respects Low Power Mode

### Android
- Downloads stored in app data
- Not included in automatic backups
- Survives app updates
- Respects Data Saver mode

### Web App
- Limited offline support
- Uses browser cache
- Service worker for core features
- Clears when cache cleared

## Data Usage Tips

### Minimize Data Usage
1. Download on Wi-Fi only
2. Limit offline storage size
3. Download essentials only
4. Use Wi-Fi for document viewing

### Optimize for Travel
1. Download all needed docs before trip
2. Enable airplane mode to prevent sync attempts
3. Work offline throughout trip
4. Sync when connected to Wi-Fi
