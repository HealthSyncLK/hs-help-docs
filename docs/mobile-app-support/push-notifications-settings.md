# Push Notifications Settings

Customize your notification preferences to stay informed.

## Notification Types

### Document Notifications
- Upload complete
- Upload failed
- Document shared with you
- Share link accessed
- Document expiration reminder

### AI Analysis
- Analysis complete
- Analysis failed
- New insights available
- Critical findings detected

### Family Activity
- Family member added
- Access permission changed
- New document added for family member
- Family member removed

### Account & Security
- Login from new device
- Password changed
- Two-factor authentication enabled
- Suspicious activity detected
- API key created/revoked

### Subscription & Billing
- Payment successful
- Payment failed
- Subscription renewed
- Subscription expiring soon
- Plan upgraded/downgraded

### System Updates
- New features available
- Scheduled maintenance
- Service updates
- Important announcements

## Managing Notifications

### Mobile Apps

#### iOS
1. Open app Settings
2. Tap "Notifications"
3. Toggle categories on/off
4. Adjust delivery settings

Or: iOS Settings > Notifications > HealthSync

#### Android
1. Open app Settings
2. Tap "Notifications"
3. Manage by category
4. Set importance levels

Or: Android Settings > Apps > HealthSync > Notifications

### Web Application

Configure browser notifications:
1. Go to Settings > Notifications
2. Click "Enable Browser Notifications"
3. Allow in browser prompt
4. Customize preferences

## Notification Preferences

### Per-Category Settings

**Document Notifications**
- All uploads
- Failed uploads only
- Sharing activity
- None

**AI Analysis**
- All results
- Abnormal findings only
- Failed analysis only
- None

**Family Activity**
- All activity
- My activity only
- Critical changes only
- None

### Delivery Method

Choose how to receive notifications:
- Push notifications ✅
- Email ✅
- SMS (Premium only)
- In-app only

### Quiet Hours

Set times to suppress notifications:
- Start time: 10:00 PM
- End time: 7:00 AM
- Days: Every day / Weekdays / Weekends
- Emergency bypass: Critical alerts only

### Notification Frequency

**Instant**
- Receive as they happen
- Recommended for important updates

**Digest**
- Batch notifications
- Options: Hourly, Daily, Weekly
- Reduces interruptions

**Important Only**
- Critical notifications only
- Errors and security alerts
- Recommended for "do not disturb" mode

## Troubleshooting

### Not Receiving Notifications

**Mobile**
1. Check app notification settings
2. Verify system permissions granted
3. Ensure "Do Not Disturb" is off
4. Check notification categories enabled
5. Restart app or device

**Web**
1. Check browser permissions
2. Ensure notifications allowed for site
3. Check browser settings
4. Try different browser
5. Clear cache and retry

### Too Many Notifications

**Solutions**
- Enable quiet hours
- Switch to digest mode
- Disable less important categories
- Use "Important Only" mode
- Adjust frequency settings

### Delayed Notifications

**Possible causes**
- Battery optimization settings
- Poor internet connection
- App restrictions
- Background data disabled

**Solutions**
- Disable battery optimization for HealthSync
- Check internet connection
- Allow background data
- Keep app updated

## Best Practices

1. **Enable critical notifications** (security, payments)
2. **Set quiet hours** for uninterrupted sleep
3. **Use categories** to filter what's important
4. **Try digest mode** to reduce interruptions
5. **Review settings periodically** as needs change

## Privacy Considerations

- Hide sensitive info from lock screen
- Use "When Unlocked" preview setting
- Disable notifications in shared environments
- Clear notification history regularly
