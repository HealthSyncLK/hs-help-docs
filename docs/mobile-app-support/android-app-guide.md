# Android App Guide

Complete guide to using HealthSync on Android devices.

## Download and Installation

### Requirements
- Android 8.0 (Oreo) or higher
- 50MB free storage space
- Internet connection

### Installing the App
1. Open Google Play Store
2. Search for "HealthSync"
3. Tap "Install"
4. Open app when installation completes
5. Sign in with your account

## Getting Started

### First Launch
1. Sign in or create account
2. Grant necessary permissions:
   - Storage (for document uploads)
   - Camera (for scanning documents)
   - Notifications (for alerts)
3. Complete quick tour
4. Start uploading records

## Key Features

### Document Scanning
- Tap camera icon
- Position document in frame
- Auto-capture when aligned
- Multi-page scanning support
- Auto-crop and enhance

### Quick Upload
- Tap "+" floating action button
- Choose source:
  - Take photo
  - Choose from gallery
  - Scan document
  - Browse files
- Add details and upload

### Offline Access
- Downloaded documents available offline
- Changes sync when online
- Auto-sync when connected

### Widgets
Add widgets to home screen:
- Recent records widget
- Quick upload widget
- AI analysis status widget

## Navigation

### Bottom Navigation Bar
- **Home**: Dashboard and recent activity
- **Records**: Browse all documents
- **Family**: Family member profiles
- **AI**: AI analysis results
- **More**: Settings and account

### Hamburger Menu
- Profile switcher
- Settings
- Help & Support
- About

## Android-Specific Features

### Material Design
- Native Android UI
- Dark theme support
- Adaptive icons
- Material You theme (Android 12+)

### System Integration
- Share documents from other apps
- Quick settings tile
- Notification channels
- Biometric authentication

### Notifications
- Upload complete
- AI analysis ready
- Share access alerts
- New family member added
- Low storage warnings

## Permissions

### Required Permissions
- **Storage**: Read/write documents
- **Camera**: Scan documents
- **Internet**: Sync data

### Optional Permissions
- **Biometric**: Fingerprint/face unlock
- **Notifications**: Push notifications

Manage in: Settings > Apps > HealthSync > Permissions

## Troubleshooting

### App Crashes
- Clear app cache
- Update to latest version
- Restart device
- Reinstall if persists

### Sync Issues
- Check internet connection
- Verify account login
- Force sync from settings
- Free up storage space

### Camera/Scanning Issues
- Grant camera permission
- Clean camera lens
- Ensure good lighting
- Try manual photo mode

## Performance Tips

- Enable Wi-Fi only sync for large files
- Download important documents for offline access
- Clear cache regularly
- Close unused documents
