# Data Protection & Encryption

Learn how HealthSync protects your health information with industry-leading security.

## Encryption Standards

### Data at Rest
**AES-256 Encryption**
- Military-grade encryption
- All documents encrypted
- Separate encryption keys per user
- Keys stored separately from data

**Database Encryption**
- Full database encryption
- Encrypted backups
- Secure key management
- Regular key rotation

### Data in Transit
**TLS 1.3**
- All connections encrypted
- Perfect forward secrecy
- Strong cipher suites only
- Certificate pinning on mobile

**API Security**
- HTTPS only
- Token-based authentication
- Rate limiting
- DDoS protection

## Data Retention

### Active Accounts
- Data retained indefinitely
- User controls retention
- Can delete anytime

### Inactive Accounts
- 90 days notice before deletion
- 1-year retention after notice
- Data export before deletion
- Can reactivate during retention

### Deleted Accounts
- 30-day grace period
- Permanent deletion after 90 days
- Backup tapes overwritten
- Legal holds excepted
