# Access Permissions & Sharing

Understand how to control who can access your health records.

## Permission Levels

### View Only
**What they can do:**
- View documents
- Read AI summaries
- See metadata (dates, categories)
- Search within shared records

**What they cannot do:**
- Edit or delete documents
- Share with others
- Run AI analysis
- Change permissions

**Best for:**
- Healthcare consultations
- Second opinions
- Family viewing
- Insurance claims

### Edit Access
**What they can do:**
- Everything in "View Only"
- Edit document metadata
- Add tags and notes
- Upload new documents
- Run AI analysis

**What they cannot do:**
- Delete documents
- Share with others
- Change permissions
- Remove access

**Best for:**
- Primary care providers
- Trusted family members
- Medical coordinators

### Full Access
**What they can do:**
- Everything in "Edit Access"
- Delete documents
- Manage sharing
- Change permissions
- Remove access

**What they cannot do:**
- Delete account
- Change subscription
- Access billing information

**Best for:**
- Caregivers
- Healthcare proxy
- Trusted partners
- Account co-managers

## Sharing Methods

### Secure Share Links

**How it works:**
1. Select documents to share
2. Generate secure link
3. Set expiration and permissions
4. Share link via email/message
5. Recipients access without account

**Link Settings:**
- Expiration date (1 day to 1 year, or never)
- Password protection (optional)
- Download permissions
- View tracking
- Revocable anytime

**Use cases:**
- Quick sharing with providers
- Insurance claims
- Legal proceedings
- Temporary access

### Email Sharing

**How it works:**
- Direct email from HealthSync
- Recipient gets notification
- Attachments or secure links
- Automatic encryption

**Options:**
- Send as attachments
- Send as secure links
- Include AI summary
- Add personal message

### Family Member Access

**How it works:**
- Add family member to account
- Set permission level
- They access via profile switcher
- Or separate login (optional)

**Controls:**
- Per-record permissions
- Category-level access
- Time-limited access
- Activity notifications

### Healthcare Provider Portal

**How it works:**
- Provider has verified account
- You grant access via email
- They access via provider portal
- Can add notes and prescriptions

**Features:**
- Two-way communication
- Provider annotations
- Visit notes
- Prescription management

## Managing Active Shares

### View All Shares
Settings > Sharing > Active Shares

**Information shown:**
- Who has access
- What they can access
- Permission level
- Expiration date
- Last accessed
- Number of views

### Modify Shares
- Extend expiration
- Change permissions
- Add/remove documents
- Revoke access

### Revoke Access
Instant revocation:
1. Open active shares
2. Select share to revoke
3. Click "Revoke Access"
4. Confirm action

Effect is immediate:
- Links stop working
- Portal access removed
- Downloads no longer accessible
- User notified (optional)

## Sharing Notifications

### For You
Receive notifications when:
- Someone accesses your records
- Share link is clicked
- Documents are downloaded
- Permissions change
- Access expires

Configure: Settings > Notifications > Sharing

### For Recipients
Recipients notified when:
- You share documents
- Access is revoked
- Permissions change
- Documents are updated

## Access Tracking

### Audit Logs
View complete history:
- Who accessed what
- When they accessed it
- What actions they performed
- IP addresses
- Device information

Access: Settings > Security > Access Logs

### Download Tracking
Track document downloads:
- When downloaded
- By whom
- From which device
- Number of downloads

## Privacy Controls

### Sensitive Information

**Redaction**
- Manually redact before sharing
- Built-in redaction tool
- Permanent or temporary

**Selective Sharing**
- Share specific pages only
- Hide sensitive sections
- Create filtered views

### Consent Management

**Explicit Consent**
- Required for adult family members
- Digital signature
- Consent records stored
- Revocable anytime

**Minor Consent**
- Parental consent
- Guardian authorization
- Age-appropriate controls

## Sharing Best Practices

### ✅ Do's
- Use expiration dates
- Enable password protection for sensitive docs
- Review active shares regularly
- Use appropriate permission levels
- Track access logs
- Revoke when no longer needed

### ❌ Don'ts
- Don't share account credentials
- Don't use public links for sensitive docs
- Don't grant full access unnecessarily
- Don't forget to set expiration
- Don't share more than needed
- Don't ignore access notifications

## Special Scenarios

### Emergency Access

**Emergency Contacts**
- Designate emergency contacts
- Automatic access in emergencies
- Define what "emergency" means
- Time-limited access

**Setup**: Settings > Emergency Access

### Healthcare Proxy

**Legal Healthcare Proxy**
- Upload legal documentation
- Grant decision-making access
- Override normal restrictions
- Complies with state laws

**Requirements:**
- Scanned power of attorney
- Notarized documents
- Verification process

### Temporary Caregivers

**Short-term Access**
- Set specific date ranges
- Limited permissions
- Automatic expiration
- No permanent access

**Use cases:**
- Traveling nurses
- Respite care
- Temporary guardians

## Compliance

### HIPAA-Compliant Sharing
- Minimum necessary principle
- Audit trails
- Encrypted transmission
- Access controls
- User accountability

### Legal Considerations
- Complies with state laws
- Respects advance directives
- Honors healthcare proxies
- Maintains audit trails for litigation

## Troubleshooting

### Share Link Not Working
- Check if expired
- Verify password (if set)
- Ensure not revoked
- Try incognito mode
- Clear browser cache

### Permission Denied
- Verify permission level
- Check if access removed
- Confirm account active
- Contact share owner

### Cannot Revoke Access
- Refresh the page
- Check internet connection
- Sign out and back in
- Contact support
