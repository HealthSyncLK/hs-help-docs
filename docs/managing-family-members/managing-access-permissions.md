# Managing Access Permissions

Control who can access and manage family member health records.

## Permission Levels

### Full Access
- View, edit, and delete records
- Manage sharing settings
- Upload new documents
- Run AI analysis

### Limited Access
- View records only
- Cannot edit or delete
- No sharing capabilities

### Custom Access
- Configure specific permissions
- Set record type restrictions
- Time-limited access

## Setting Permissions

1. Go to family member profile
2. Click "Manage Permissions"
3. Select permission level
4. Save changes

## Guardian Controls

Special permissions for parents/guardians:
- Override sharing restrictions
- Monitor all access
- Receive activity notifications
- Age-gated content filtering

## Access Expiration

Set expiration dates for temporary access:
- Specify end date
- Automatic revocation
- Renewal notifications
