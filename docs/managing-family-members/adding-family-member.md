# Adding a Family Member

Learn how to add family members to your HealthSync account.

## Prerequisites

- Active HealthSync account
- Consent from the family member (if applicable)
- Family member's basic information

## Step-by-Step Guide

1. **Navigate to Family Section**
   - Click on "Family Members" in the main menu
   - Select "Add New Family Member"

2. **Enter Basic Information**
   - Full name
   - Date of birth
   - Relationship to you
   - Email address (optional)

3. **Set Permissions**
   - Choose access level
   - Configure sharing permissions
   - Set up notifications

4. **Create Profile**
   - Review information
   - Confirm and create profile

## Age Considerations

### Minor Children (Under 18)
- Full guardian access by default
- Automatic parental controls
- Age-appropriate consent settings

### Adult Family Members
- Requires their consent
- They receive an invitation email
- Can request access removal
