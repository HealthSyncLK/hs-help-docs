# Switching Between Profiles

Easily navigate between different family member profiles.

## Profile Switcher

### Desktop/Web
- Click profile name in top navigation bar
- Select from dropdown menu
- Switch instantly to selected profile

### Mobile App
- Tap profile icon
- Swipe through family members
- Select desired profile

## Quick Switch Features

- Recently viewed profiles appear first
- Filter by family member name
- Visual indicators for unread notifications
- Quick access shortcuts

## Profile Context

When switching profiles:
- Dashboard updates to show selected member's data
- All records filtered to that profile
- AI analysis history specific to that member
- Sharing settings apply per profile

## Keyboard Shortcuts

Desktop shortcuts for quick switching:
- `Ctrl + 1` through `Ctrl + 9` for first 9 profiles
- `Alt + P` to open profile switcher
