# Common Error Messages

Understand and resolve common error messages in HealthSync.

## Upload Errors

### "File size too large"
**Cause**: File exceeds plan limit  
**Solution**: 
- Compress the file
- Upgrade to Premium for larger limits
- Split multi-page documents

### "Unsupported file format"
**Cause**: File type not supported  
**Solution**: 
- Convert to PDF, JPG, or PNG
- Check supported formats list

### "Upload failed - network error"
**Cause**: Internet connection issue  
**Solution**: 
- Check your internet connection
- Retry the upload
- Try different network if available

## AI Analysis Errors

### "Analysis failed - unable to extract text"
**Cause**: Poor image quality or unreadable document  
**Solution**: 
- Re-upload with better quality
- Use PDF instead of image
- Ensure text is not handwritten

### "Analysis quota exceeded"
**Cause**: Monthly AI analysis limit reached  
**Solution**: 
- Wait until next billing cycle
- Upgrade to higher plan
- Prioritize important documents

## Login Errors

### "Invalid email or password"
**Cause**: Incorrect credentials  
**Solution**: 
- Double-check email and password
- Use "Forgot Password" to reset
- Check for caps lock

### "Account locked"
**Cause**: Multiple failed login attempts  
**Solution**: 
- Wait 15 minutes
- Use password reset
- Contact support if persists

## Sharing Errors

### "Share link expired"
**Cause**: Time-limited link has expired  
**Solution**: 
- Request new share link
- Check expiration date
- Generate permanent link

### "Permission denied"
**Cause**: Insufficient access rights  
**Solution**: 
- Check your permission level
- Contact account owner
- Request elevated access
