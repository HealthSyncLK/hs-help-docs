# Search & Filter Help

Get better search results and learn to use advanced filters.

## Basic Search

### Quick Search
- Use the search bar at the top of any page
- Type keywords, document names, or medical terms
- Results appear instantly as you type

### Search Tips
- Use specific terms for better results
- Include dates if known
- Try alternate spellings of medical terms
- Use quotation marks for exact phrases

## Advanced Search

### Search Operators

**AND** - Find records with all terms
```
diabetes AND medication
```

**OR** - Find records with any term
```
"blood test" OR "lab results"
```

**NOT** - Exclude terms
```
scan NOT MRI
```

**Quotes** - Exact phrase
```
"high blood pressure"
```

### Date Range Search
- Click "Date Range" filter
- Select start and end dates
- Or use predefined ranges (last week, month, year)

## Filters

### Available Filters

**Document Type**
- Lab Results
- Prescriptions
- Imaging
- Visit Notes
- Vaccination Records
- Other

**Date Added**
- Today, This Week, This Month
- Custom date range

**Family Member**
- Filter by specific person
- View all or individual profiles

**Tags**
- Filter by custom tags
- Combine multiple tags

**AI Analysis Status**
- Analyzed
- Not analyzed
- Analysis failed

**Shared Status**
- Currently shared
- Not shared
- Expired shares

### Applying Multiple Filters
- Select multiple filters simultaneously
- Results refined with each added filter
- See active filters at top of results
- Clear individual filters or all at once

## Saving Searches

### Save Frequently Used Searches
1. Perform your search with filters
2. Click "Save Search"
3. Name your saved search
4. Access from "Saved Searches" menu

### Managing Saved Searches
- Edit search criteria
- Rename saved searches
- Delete unused searches
- Set as default view

## Search Troubleshooting

### No Results Found

**Common issues:**
- Spelling errors
- Too specific search terms
- Filters too restrictive
- Document not yet indexed

**Try:**
- Simplify search terms
- Remove some filters
- Check spelling
- Wait a few minutes for new uploads

### Too Many Results

**Narrow down with:**
- Add more specific terms
- Apply date range filter
- Filter by document type
- Use advanced operators
