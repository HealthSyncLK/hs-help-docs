# Sync Problems

Troubleshoot data synchronization issues between devices.

## Common Sync Issues

### Changes Not Appearing on Other Devices

**Possible causes:**
- Device not connected to internet
- Sync disabled in settings
- Outdated app version
- Cache issues

**Solutions:**
1. Check internet connection
2. Force manual sync
3. Update app to latest version
4. Clear app cache

### Manual Sync

#### Web App
1. Click profile icon
2. Select "Sync Now"
3. Wait for completion notification

#### Mobile App
1. Pull down on home screen
2. Release to trigger sync
3. Watch for sync indicator

## Sync Conflicts

### What Are Sync Conflicts?
Occur when same record edited on multiple devices simultaneously.

### Resolving Conflicts
- HealthSync shows both versions
- Choose which version to keep
- Or merge changes manually
- Save preferred version

## Upload Not Syncing

### Check Upload Status
1. Go to **Activity** log
2. View "Pending Uploads"
3. Check for errors or warnings

### Force Upload Retry
- Select pending upload
- Click "Retry"
- Ensure stable connection

## Mobile App Sync Settings

### Configure Sync Preferences

**Auto-Sync**
- Enable/disable automatic syncing
- Set sync frequency (instant, hourly, daily)
- Background sync options

**Wi-Fi Only**
- Sync only on Wi-Fi to save data
- Recommended for large files
- Override for urgent uploads

**Battery Optimization**
- Reduce sync frequency on low battery
- Disable background sync to save power

## Offline Mode

### Working Offline
- View and edit documents without connection
- Changes queued for upload
- Sync automatically when online

### Offline Limitations
- Cannot run AI analysis
- Cannot share documents
- Cannot add new family members
