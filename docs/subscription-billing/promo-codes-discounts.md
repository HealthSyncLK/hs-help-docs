# Promo Codes & Discounts

Learn about available discounts and how to apply promo codes.

## Available Discounts

### Annual Billing
Save 20% by choosing annual billing instead of monthly.

### Student Discount
50% off any plan with valid student email address.
- Must verify .edu email
- Renewable annually

### Healthcare Worker Discount
30% off for verified healthcare professionals.
- Requires ID verification
- Includes medical students

### Family Plan Discount
Add family members at reduced rates.
- 2nd member: 25% off
- 3rd+ members: 50% off

### Nonprofit Organizations
Special pricing available for registered nonprofits.
- Contact sales for custom quote

## Applying Promo Codes

### During Sign-Up
1. Choose your plan
2. Click "Have a promo code?"
3. Enter code
4. Discount applies automatically

### Existing Subscriptions
1. Go to **Settings** > **Billing**
2. Click **Apply Promo Code**
3. Enter code
4. Click **Apply**
5. Discount reflected in next billing cycle

## Promo Code Terms

- One promo code per account
- Cannot combine with other offers (unless specified)
- Some codes have expiration dates
- Discounts may apply to first billing cycle only
